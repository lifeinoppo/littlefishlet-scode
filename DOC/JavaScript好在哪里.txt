JavaScript 好在哪里呢？
根据JavaWorld的Martin Heller的说法， Javascript不仅仅可以搭配
Html5 和 CSS 来构建Web的前端，在移动应用中也能看到他的身影。
甚至在后端开发领域，他也通过Node.js服务器在发挥作用。
如果你读一下 《全栈Javascript介绍》，你就会理解Javascript
远远不止是一门 90秒可以掌握 （哇喔，这说法可真诱人，相信我，这不是真的）的编程
语言这么简单。
Js （Javascript简称）在产品经理和设计师人群中开始流行，因为
他实现了很多具有良好用户体验的功能，支持大多数流行的浏览器和平台。
